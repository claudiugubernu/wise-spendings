# Wise Spendings

Your smart budget app companion.

<img width="926" alt="Screenshot 2022-09-23 at 18 18 31" src="https://user-images.githubusercontent.com/55931383/192023854-637f43e1-6917-4493-804c-ff869b759328.png">

A Web Application that allows you to keep your spendings in check.

## v0.1 Features
- Light/Dark Mode
- Language Picker ( En & Ro )
- Currency Picker ( GBP, USD, EUR, RON )

<img width="581" alt="Screenshot 2022-09-23 at 18 18 47" src="https://user-images.githubusercontent.com/55931383/192024325-ed9a154a-f8c0-46c1-b952-cc8c2bfbe29b.png">

You can Add/Delete Budgets, Spendings and have a clear view of what your spendings are. 

<img width="499" alt="Screenshot 2022-09-23 at 18 19 31" src="https://user-images.githubusercontent.com/55931383/192024473-d994c860-545a-4905-a29c-b88494ec9362.png">

This Web Application also has a color system to easy indicate how your budget looks like: 
- Blue is shown if your spendings are less than 50% of your total budget
- Yellow if your spendings are between 50% and 75% of your total budget
- Red if your spendings are over 75% of your total budget

You can also add uncategorized, singled spendings or if you remove a budget it will automatically take the spendings made and add them in a Uncategorized card to keep an accurate record of your spending.

## Technologies used

React, HTML5, SASS

## Installation

Clone the repo or download the .zip and add to your project

Install NPM

```bash
npm install
```

## Running commands

To start the local server, sass compiling and js watch simply run:

```bash
npm start
```
